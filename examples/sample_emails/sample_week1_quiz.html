Question 1 · Day 1 · What is a Model?
Which best describes what a machine learning model is?
A. A database that stores predictions
B. A function with parameters learned from data
C. A pipeline that cleans data
D. A neural network with hidden layers
Question 2 · Day 2 · Supervised vs Unsupervised

A. Supervised learning
B. Unsupervised learning
C. Reinforcement learning
D. Semi-supervised learning
Question 3 · Day 3 · Train/Val/Test
What is data leakage?
A. Disk space running out during training
B. When test set information influences training
C. Sending data to the wrong cloud region
D. Forgetting to back up your dataset
Question 4 · Day 4 · Bias-Variance
A model gets 99 percent on training but 60 percent on test. This is:
A. High bias (underfitting)
B. High variance (overfitting)
C. Data leakage
D. Class imbalance

Answers

Q1: A function with parameters learned from data
A model is a parameterized function. Training is the process of choosing parameters that fit the data. Neural networks are one specific type of model, not the definition.
Q2: Unsupervised learning
No labels + finding structure = unsupervised. Specifically, this is a clustering problem.
Q3: When test set information influences training
Data leakage is when info you should not have at train time sneaks in, causing inflated metrics and brittle production models.
Q4: High variance (overfitting)
Memorizing training data but failing on new data is the textbook overfitting signature. Add regularization or more data.
