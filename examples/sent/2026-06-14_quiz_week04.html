Question 4 · Day 14 · Optimizers: SGD, Momentum, Adam
What is the primary advantage of using the Adam optimizer?
A. It converges faster than SGD
B. It is more robust to hyperparameter tuning
C. It adapts the learning rate for each parameter individually
D. It is more computationally efficient

Answers

Q1: Ability to handle longer sequences
LSTMs are designed to handle the vanishing gradient problem in traditional RNNs, allowing them to effectively learn from longer sequences. This makes LSTMs particularly useful for tasks like language modeling and time series forecasting.
Q2: To extract local features
The convolutional layer uses filters to scan the input data, extracting local features that are useful for tasks like image classification. This allows the CNN to learn hierarchical representations of the input data.
Q3: It sets some weights to zero, while reducing others
L1 regularization, also known as Lasso regularization, adds a term to the loss function that is proportional to the absolute value of the model weights. This can cause some weights to be set to zero, effectively performing feature selection, while reducing the magnitude of others.
Q4: It adapts the learning rate for each parameter individually
The Adam optimizer uses a combination of the momentum and RMSProp algorithms to adapt the learning rate for each parameter based on the magnitude of the gradient. This helps to stabilize the training process and improve convergence.
