Question 4 · Day 19 · Word Embeddings: Word2Vec and GloVe
What is the primary difference between Word2Vec and GloVe word embeddings?
A. Word2Vec uses a neural network architecture, while GloVe uses a matrix factorization approach
B. Word2Vec uses a matrix factorization approach, while GloVe uses a neural network architecture
C. Word2Vec captures semantic relationships, while GloVe captures syntactic relationships
D. Word2Vec captures local context, while GloVe captures global context

Answers

Q1: To split text into individual words or subwords
Tokenization is a crucial step in NLP that involves breaking down text into individual words or subwords, which can then be used as input for downstream tasks. This process allows models to process and understand the text at a more granular level.
Q2: Encoder-Decoder
Encoder-decoder models are widely used for sequence-to-sequence tasks such as language translation, where the encoder processes the input sequence and the decoder generates the output sequence. This architecture allows for efficient processing of input and output sequences.
Q3: To preserve the order of input sequences
Positional encoding is used to preserve the order of input sequences in transformer models, which do not have recurrent or convolutional structures to capture sequential relationships. By adding positional encodings to the input embeddings, the model can capture the order of the input sequence.
Q4: Word2Vec uses a neural network architecture, while GloVe uses a matrix factorization approach
Word2Vec and GloVe are two popular word embedding techniques that differ in their approach to learning word representations. Word2Vec uses a neural network architecture to learn word embeddings, while GloVe uses a matrix factorization approach to capture word co-occurrence patterns.
