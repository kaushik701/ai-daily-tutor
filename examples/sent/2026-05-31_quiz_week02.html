A model with high bias and low variance is likely to be

C. Regularly fit, with balanced performance on the training and test sets

Question 4 · Day 2 · Supervised vs Unsupervised vs Reinforcement Learning
Which type of machine learning involves an agent learning to take actions in an environment to maximize a reward?
A. Supervised learning
B. Unsupervised learning
C. Reinforcement learning
D. Semi-supervised learning

Answers

Q1: To find the optimal parameters for a model
Gradient descent is an optimization algorithm used to minimize the loss function by adjusting the model's parameters, thereby finding the optimal parameters for the model.
Q2: Mean squared error
Mean squared error is a common choice for regression problems because it measures the average squared difference between predicted and actual values, providing a clear indication of the model's performance.
Q3: Underfitting, with poor performance on both the training and test sets
A model with high bias and low variance is likely to be underfitting, as it is too simple to capture the underlying patterns in the data, resulting in poor performance on both the training and test sets.
Q4: Reinforcement learning
Reinforcement learning is a type of machine learning where an agent learns to take actions in an environment to maximize a reward, through trial and error, and receiving feedback in the form of rewards or penalties.
